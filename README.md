# HNOptimizer-4.1
Herramienta avanzada de optimización para Windows enfocada en rendimiento y limpieza de sistema.

Deep System Junker: Limpieza profunda de más de 100 rutas de archivos temporales y basura del sistema.

Registry Tweaks: Optimización de latencia y priorización de procesos mediante inyección de registros (HKLM/HKCU).

RAM Management: Liberación de memoria física en tiempo real mediante el análisis de procesos con psutil.

Multi-threading Engine: Ejecución asíncrona de tareas para mantener la interfaz fluida y evitar bloqueos.

Nota sobre los Antivirus: Debido a que HN OPTIMIZER realiza cambios avanzados en el Registro de Windows y requiere permisos de administrador para limpiar carpetas protegidas, algunos motores de antivirus basados en IA (como los de VirusTotal) pueden marcar el archivo como sospechoso.
